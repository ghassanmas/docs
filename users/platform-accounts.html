# Accounts

The wbstack platform requires a user account to use.

# Creation

TIP

You need a invite code before you can create an account! If you don't have one, contact Addshore using this form (opens new window) or on Twitter (opens new window).

In order to create an account you need to hit the GET STARTED button in the top right.

This will take you to the account creation page (opens new window).

Enter your details and your invite code into the form and click CREATE ACCOUNT.

You should be redirected to the dashboard page.

WARNING

At this stage you still need to verify your email address before continuing to make a site. See Account Verification

# Verification

Head to your inbox and look for the email contining the verification link.

It should look something like this.

Click the link, and your account should become verified.

Returning to the dashboard you should now see a button allowing you to create a Wiki.

# Management

Not much account management is currently implemented.

You can view your registered email address on the user page (opens new window).

If you have forgotten your password you can use the forgotten password page (opens new window).
